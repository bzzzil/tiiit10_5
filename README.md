# Array Reverser

Программа на C++ для обращения (переворота) массива - изменения порядка элементов на обратный.

## Функционал
- Переворачивает одномерный массив целых чисел
- Использует эффективный алгоритм с двумя указателями
- Работает "на месте" (in-place) без создания дополнительного массива
- Автоматически определяет размер массива
