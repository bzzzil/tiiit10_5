# Программа для обращения (реверсирования) массива
## Описание
Эта программа на C++ переворачивает элементы массива в обратном порядке. Исходный массив изменяется "на месте" (in-place) без создания дополнительного массива.

## Функциональность
- Принимает исходный массив целых чисел

- Реверсирует порядок элементов в массиве

- Выводит результат на экран

## Пример работы
Входные данные: {1, 2, 3, 4, 5}  
Выходные данные: {5, 4, 3, 2, 1}
